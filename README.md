# Deep Learning Specialization: Study Notes & Reference

Welcome! This repository contains my personal study notes and comprehensive summaries for the **Deep Learning Specialization** series by **Andrew Ng**, offered by **DeepLearning.AI** on **Coursera**.

## About These Notes
The purpose of this repository is to provide a structured reference for students and practitioners to quickly review key concepts, mathematical derivations, and architectural details covered in the specialization.

> [!IMPORTANT]
> **Not a Replacement:** These notes are designed to supplement your learning. They are **not** a substitute for the actual lectures, assignments, and quizzes provided by the Coursera platform. To gain the full benefit of the curriculum, I highly recommend enrolling in the [Deep Learning Specialization](https://www.coursera.org/specializations/deep-learning).

## Curriculum Breakdown

Click the links below to access the detailed notes for each course:

    * Fundamentals of Logistic Regression, Vectorization, and Hidden Layers.

    * Hyperparameter Tuning, Regularization (L2, Dropout), and Optimization (RMSprop, Adam).
3.  **[Structuring Machine Learning Projects](./course-3/notebooks/)**
    * Strategies for Error Analysis, Orthogonalization, and Mismatched Data Distributions.
4.  **[Convolutional Neural Networks](./course-4/notebooks/)**
    * Foundations of CNNs, Object Detection (YOLO), Face Recognition, U-Net, Inception, many more.
5.  **[Sequence Models](./course-5/notebooks/)**
    * RNNs, GRUs, LSTMs, and the Transformer Architecture (Attention Mechanisms).
